## Installation & Usage

1. **Install Dependencies**:
   ```bash
   npm install
   ```

2. **Start the Bridge**:
   ```bash
   node server.mjs
   ```
   *Runs on port 8787 by default.*

## Architecture
- **Server**: Node.js + Express + WebSocket
- **Bridge**: CDP (Chrome DevTools Protocol) to IDE port 9000/9222
- **Frontend**: Vanilla HTML/JS (Mobile Optimized)
